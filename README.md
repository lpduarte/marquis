# Marquis

*Devolve o monospace às máquinas.*

Um leitor simples de markdown. Não edita, não sincroniza, não sugere.

**→ [lpduarte.github.io/marquis](https://lpduarte.github.io/marquis/)**

## O que faz

Lê ficheiros `.md`, `.markdown` e `.txt` numa interface simples e limpa, com três temas: claro, sépia e escuro.

## O que não faz

- Não edita.
- Não sincroniza, não envia ficheiros, não tem conta.
- Os ficheiros nunca saem do browser.

## Desenvolvimento

Sem build step, sem dependências a instalar. É HTML, CSS e JavaScript servidos como ficheiros estáticos.

## Créditos

- [marked](https://github.com/markedjs/marked) — parser de markdown
- [DOMPurify](https://github.com/cure53/DOMPurify) — sanitização de HTML
- [Literata](https://fonts.google.com/specimen/Literata) — tipografia
